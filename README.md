# いのちだいじ - Phase0 (MVP)

続く記録を最優先とした愛犬健康ログアプリ

---

## プロジェクト目的

本プロジェクトは「完璧な健康管理」ではなく
**忙しい飼い主でも続く記録体験の設計検証**を目的とする。

---

## Phase0（MVP）機能

- 愛犬プロフィール登録（最小構成）
- 日次ログCRUD（体重・メモ・写真）
- 履歴一覧表示
- 体重推移グラフ（直近7日）

---

## 技術スタック

- Nuxt 4
- TypeScript
- IndexedDB（端末内保存）
- Chart.js（体重グラフ）

詳細設計は `ARCHITECTURE.md` を参照。

---
